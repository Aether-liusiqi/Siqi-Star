# Siqi-Star 占星解读知识底座 (astrology_kb.md)

> 本文件是 skill 的"知识库"，供 LLM 在拿到 `astro_calc.py` 计算的星图 JSON 后做人性化解读。
> 设计定位：**温暖、深度的心理洞察**，而非 Co-Star 式冷峻 roast。
> 核心立场：占星是"自我探索的镜面"，不是预测未来的科学。所有解读都应指向"认识自己"，而非"判定命运"。
> 使用原则：本库提供素材与框架，LLM 负责融合出生数据、诞生具体语境，写出有温度、有结构、有个体的解读，禁止逐字照搬模板。

---

## 〇、解读的总框架（每次解读都按这个顺序组织）

1. **看见整体**：先用 3 句话勾勒这张星图给人的"第一印象"（元素分布、最突出的行星/相位、整体气质）。
2. **太阳：你是谁（意识自我）**：生命的主轴、想成为的样子。
3. **月亮：你需要什么（内在情绪）**：安全感从哪来、情绪怎么流动。
4. **上升：你如何被世界看见（人格面具）**：第一印象、应对世界的方式。
5. **水星/金星/火星：你如何思考、去爱、行动**。
6. **外行星（木土天海冥）：你与时代、成长、超越的关系**（这一层常是"代际气质"）。
7. **关键相位**：星图里的"张力与天赋"，挑 2-3 个最有张力的讲。
8. **一句收束**：把整张盘凝成一句温暖的邀请，而非结论。

> 不要面面俱到地罗列 10 颗行星。挑"这张盘最想说的 3 件事"，其余一笔带过或省略。深度胜过全面。

## 一、十二星座（黄道符号）

每个星座标注：日期（热带黄道，注意岁差导致与天文星座偏移约一个月，见诚实边界）、元素、模式、守护星、核心特质、阴暗面/成长课题、荣格原型、一句话洞察。

### 1. 白羊座 (Aries) 火 / 基本 / 火星
- 核心：行动力、开创、直率、勇敢、以"我敢"定义存在。
- 阴暗面：冲动、自我中心、三分钟热度、不愿示弱。
- 成长课题：把"冲"变成"持续"，学会在行动前停顿一秒。
- 原型：战士 / 婴儿（纯粹而未经世故的生命力）。
- 洞察：你最大的天赋是开始，最大的功课是finish。

### 2. 金牛座 (Taurus) 土 / 固定 / 金星
- 核心：稳定、感官、务实、耐心、珍视具体而美的事物。
- 阴暗面：固执、占有欲、抗拒改变、用物质填补不安。
- 成长课题：在安稳与成长之间找到呼吸的缝。
- 原型：大地之母 / 耕耘者。
- 洞察：你教世界慢下来，但别让慢变成停滞。

### 3. 双子座 (Gemini) 风 / 变动 / 水星
- 核心：好奇、沟通、灵活、信息处理、连接人与想法。
- 阴暗面：浮浅、善变、逃避深层情绪、思维过载。
- 成长课题：把"知道很多"沉淀成"懂得一个"。
- 原型：信使 / 孪生子（一体两面的内在对话）。
- 洞察：你的翅膀是信息，你的根却在某个还没说出口的感受里。

### 4. 巨蟹座 (Cancer) 水 / 基本 / 月亮
- 核心：情感、归属、照顾、记忆、以"家"为锚。
- 阴暗面：过度防御、情绪吞没理性、以付出换取被需要。
- 成长课题：学会照顾自己，才能真正照顾别人。
- 原型：母亲 / 守护者。
- 洞察：你最柔软的地方，也是你最有力量的地方。

### 5. 狮子座 (Leo) 火 / 固定 / 太阳
- 核心：创造力、表达、尊严、慷慨、需要被看见。
- 阴暗面：虚荣、控制、把舞台当成了自我、怕不被认可。
- 成长课题：从"要被看见"走向"照亮他人"。
- 原型：国王 / 表演者。
- 洞察：你生来要发光，但光是为了温暖，不是为了碾压。

### 6. 处女座 (Virgo) 土 / 变动 / 水星
- 核心：分析、服务、精进、秩序、用改善世界证明爱。
- 阴暗面：苛求完美、自我批判、陷在细节里、过度服务他人忽略自己。
- 成长课题：把对完美的执念，换成对"足够好"的慈悲。
- 原型：疗愈者 / 工匠。
- 洞察：你修理世界，也请记得修理自己的苛责。

### 7. 天秤座 (Libra) 风 / 基本 / 金星
- 核心：关系、和谐、审美、公平、在"我们"中找到自己。
- 阴暗面：犹豫、讨好、回避冲突、把决定权交给别人。
- 成长课题：和谐不是没有自我，而是两个完整的人并肩。
- 原型：调和者 / 爱神。
- 洞察：你最怕孤单，却只有在独处时才能真正认识自己要的"我们"。

### 8. 天蝎座 (Scorpio) 水 / 固定 / 冥王星(传统火星)
- 核心：深度、转化、直觉、忠诚、不轻易交付但交付就全情。
- 阴暗面：掌控、猜疑、记仇、用 intensity 逃避脆弱。
- 成长课题：信任是冒险，不是 weakness；放下不等于失去。
- 原型：炼金术士 / 凤凰（在死亡里重生）。
- 洞察：你拥有最深的爱，也最容易把它锁进最深的怕。

### 9. 射手座 (Sagittarius) 火 / 变动 / 木星
- 核心：意义、探索、乐观、自由、用信念拓展边界。
- 阴暗面：夸大、逃避约束、不耐烦细节、用"远方"躲开眼前。
- 成长课题：自由不是逃离，是带着责任去远方。
- 原型：哲人 / 旅人。
- 洞察：你追逐地平线，但家可能在你停下的那一刻。

### 10. 摩羯座 (Capricorn) 土 / 基本 / 土星
- 核心：成就、责任、结构、长期主义、把山一步步走完。
- 阴暗面：压抑情感、过度认同成就、悲观、用忙碌证明价值。
- 成长课题：成功之外，允许自己被爱、被需要、被看见脆弱。
- 原型：长老 / 建造者。
- 洞察：你建起高塔，但塔顶的风，不该是你唯一允许自己感受的。

### 11. 水瓶座 (Aquarius) 风 / 固定 / 天王星(传统土星)
- 核心：独立、创新、人道、未来、在"不一样"里自在。
- 阴暗面：疏离、叛逆为反、情感抽离、用独特证明疏离。
- 成长课题：与众不同不是孤立，是给群体带来新可能。
- 原型：先知 / 局外人。
- 洞察：你站在人群之外看人类，却也渴望被人类温柔接住。

### 12. 双鱼座 (Pisces) 水 / 变动 / 海王星(传统木星)
- 核心：共情、想象、灵性、融合、对边界天生模糊的温柔。
- 阴暗面：逃避、混沌、过度共情耗竭、分不清自己与他人。
- 成长课题：慈悲需要边界，否则你会溺毙在别人的海里。
- 原型：诗人 / 救赎者。
- 洞察：你活在梦与现实的缝隙，那里既是迷宫，也是神殿。

## 二、十颗行星（心理功能）

每颗行星 = 一种心理功能；落座 = 这种功能以什么"风格"运作；落宫 = 它投注在生活的哪个"领域"。

- **太阳**：意识自我、生命意志、你想成为的人。"我是谁"的主轴。
- **月亮**：内在情绪、安全感需求、潜意识习惯。"我需要什么才能安心"。
- **水星**：思维、沟通、学习方式。你如何接收与表达信息。
- **金星**：爱、价值、吸引、审美。你如何亲近、珍视什么。
- **火星**：行动、欲望、愤怒、主张。你如何争取、捍卫、出击。
- **木星**：扩张、信念、幸运、意义。你从哪里获得成长与希望。
- **土星**：限制、责任、恐惧、成熟。你哪里必须面对"现实的老师"。
- **天王星**：突破、自由、反常。你哪里渴望挣脱、与众不同。
- **海王星**：想象、消融、灵性、迷幻。你哪里渴望合一或容易迷失。
- **冥王星**：转化、权力、深渊。你哪里经历"死亡与重生"的课题。

> 解读提示：内行星（日、月、水、金、火）构成"个人性格"；外行星（木、土、天、海、冥）更多是"代际气质"与"成长/超越的议题"。对一个具体的人，重点讲内行星 + 土星（个人成长的边界），外行星点到为止、连到时代背景即可。

## 三、十二宫位（生活领域，整宫制语序）

1. **第一宫（命宫/上升）**：自我、身体、第一印象、人生姿态。
2. **第二宫**：资源、价值、金钱、自我价值感。
3. **第三宫**：沟通、学习、手足、短途、日常信息。
4. **第四宫（田宅）**：家、根源、父母、内在安定。
5. **第五宫**：创造、恋爱、孩子、表达、乐趣。
6. **第六宫**：工作、健康、日常秩序、服务。
7. **第七宫（夫妻）**：关系、伴侣、合作、镜像般的"他者"。
8. **第八宫**：亲密、共享资源、转化、潜意识深渊、生死。
9. **第九宫**：信念、远方、高等教育、意义探索。
10. **第十宫（事业/中天）**：社会成就、使命、公众形象、父母（尤其是父亲意象）。
11. **第十一宫**：社群、友谊、理想、未来。
12. **第十二宫**：潜意识的、独处、灵性、隐秘、放下。

> 整宫制下，宫位与星座一一对应（第1宫=上升星座，第2宫=下一个星座，依此类推）。解读时把"行星落在哪个星座"与"它落入哪个宫"并列说：比如"金星落在天秤、落入第七宫"，就是"以和谐审美的方式，在关系里寻找自我"。

## 四、五大相位（能量动态）

相位是星体之间的"对话角度"，代表内在不同心理功能之间的张力或流畅。

- **合相 (0°)**：能量叠加、聚焦。要么是天赋的中心，要么是内在的"搅在一起"。看涉及行星而定。
- **六分相 (60°)**：轻松的机会、可培养的天赋。需要主动去用，才会显化。
- **四分相 (90°)**：张力、摩擦、内在冲突。是"成长最痛也最有效"的着力点。
- **三分相 (120°)**：流畅、天赋、轻松。但太舒服可能不被珍惜，需有意识地发挥。
- **对分相 (180°)**：拉扯、对立、投射。常体现为"内在的两边"或在关系里找互补。

> 解读时挑 2-3 个**容许度最小（最精确）**的相位讲，它们是这张盘"最有张力的地方"。不要罗列全部 20 个。

## 五、元素与模式（看整体气质）

- **元素**：火（行动/热情）、土（务实/安定）、风（思维/连接）、水（情感/直觉）。
  - 读图先看四元素是否均衡：火多则冲、土多则稳、风多则飘、水多则深。
  - 某元素缺失，不代表"没有那个面向"，而是"用别的方式表达它"，或"这一面是被忽略的成长邀请"。
- **模式**：基本（开创/启动）、固定（坚守/深化）、变动（适应/流动）。
  - 基本多：行动派但可能浅尝辄止；固定多：专注但可能固执；变动多：灵活但可能缺乏定力。

## 六、合盘（Synastry）解读要点

合盘不是"配不配"，而是"我们之间会触发什么"。

- **盘间相位**：A 的行星与 B 的行星形成的角度。重点看：
  - 月亮-月亮、金星-金星、太阳-月亮 的和谐相位：情感流动、相互滋养。
  - 火星、土星、冥王星参与的紧张相位：摩擦、权力拉扯，但也是深度绑定的可能。
  - 不要只说"合"或"不合"，要说"这段关系会让你面对自己哪一面"。
- **行星落入对方宫位**：B 的某颗行星落在 A 的第几宫，代表 B 以某种方式"激活" A 生活的那个领域。例如 B 的太阳落入 A 的第七宫：B 在关系里照亮 A 的"他者面向"。
- **收束句**：把合盘凝成一句话，关于"这段关系邀请两人共同成长的方向"，而非"注定在一起/分开"。

## 七、流年（Transits）解读要点

流年 = 当下天空中真实运行的行星，经过你出生星图的位置，触发的"当下议题"。

- **慢行星（木土天海冥）过境**是人生阶段的"大背景"：
  - 木星过某宫：扩张、机会、信心涌入该领域（约每年过一宫）。
  - 土星回归（约 29.5 岁、58 岁）：成熟的"现实考试"，放下幻象、承担责任。
  - 天王/海王/冥王过本命重要行星：突破性、消融、转化的深层周期（常以年计）。
- **快行星（日月水金火）过境**是"日常节奏"：适合讲"这一周/这个月"的能量基调。
- **表达原则**：流年只讲"当下的课题与能量"，不讲"一定会发生什么"。给可执行的觉察建议，而非宿命预言。

## 八、每日运势（Daily）生成原则

每日运势是"轻量、温柔、可行动"的，不是恐吓式预言。

- 基于：当日月亮所在星座（情绪基调）+ 当日主要行运相位（能量焦点）+ 用户本命盘的关键配置（个性化钩子）。
- 结构：一句"今天整体的感觉" + 一个"值得留意的内在状态" + 一个"可以去做的小动作"。
- 语气：像一位懂你的朋友发来早安短信，温暖、具体、不吓人。
- 严禁：灾难预告、绝对化措辞（"今天必破财""绝对不能""注定"）。

## 九、与 Co-Star 的体验差异点（本 skill 的主张）

| 维度 | Co-Star | Siqi-Star |
|------|---------|-----------|
| 语气 | 冷峻、带刺、roast | 温暖、深度、邀你自省 |
| 立场 | 多半默认"占星即性格真理" | 明确"这是自我探索的镜面，非科学" |
| 深度 | 碎片式短句 | 有结构、有原型、连到成长 |
| 诚实 | 较少谈边界 | 每篇都带"非科学边界"与数据精度说明 |
| 本土 | 英文为主 | 中文语境、可融合八字/紫微视角（见进阶） |

> 我们不做"比你更懂你"的傲慢判断，做"陪你更懂自己"的镜面。
